# Brother's Eye / React AI Tool

> [!WARNING]
> **THIS REPOSITORY IS NO LONGER WORKING OR MAINTAINED.**

## Description
This is a React application built using Vite, React 19, and TailwindCSS. It was originally designed as an AI tool but is no longer functional.

## Getting Started (Deprecated)

### Prerequisites
- Node.js (for running NPM commands)

### Installation
```bash
npm install
```

### Running Locally
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```
